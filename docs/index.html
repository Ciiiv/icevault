API Configuration

🔒 Your keys are private and secure
API keys are stored only in your browser's local storage on this device. They are never sent to, saved in, or accessible by this app, any server, or any database. Keys go directly from your browser to the respective API (Anthropic / eBay) and nowhere else.

Even if user accounts are added in the future, your API keys will never be saved to any account or database — by design. You will need to re-enter your keys if you clear your browser cache, use a private/incognito window, switch browsers, or use a different device.
